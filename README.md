# 📊 Mobile Sales Dashboard (Power BI)

## 📌 Overview

This project is an interactive **Power BI Dashboard** developed to analyze mobile sales performance and generate meaningful business insights through data visualization.

The dashboard provides an overview of sales, transactions, customer ratings, payment methods, and brand performance, enabling data-driven decision-making.

---

## 🎯 Problem Statement

To transform raw mobile sales data into an interactive dashboard that helps businesses monitor performance, identify trends, and support strategic decisions.

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization
* Microsoft Excel

---

## 📂 Dashboard Features

* Total Sales Analysis
* Total Quantity Sold
* Total Transactions
* Average Sales Value
* Brand-wise Sales Performance
* City-wise Sales Analysis
* Monthly Sales Trend
* Customer Rating Analysis
* Payment Method Distribution
* Interactive Filters and Slicers
